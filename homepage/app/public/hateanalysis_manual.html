Hate Measurement

1. 분석 개요

혐오도 분석은 업로드된 텍스트 데이터(CSV)에 대해 문장 단위로 혐오 표현 가능성을 분석합니다. CSV의 각 행 데이터는 여러 혐오 카테고리에 대해 확률(백분율) 형태의 점수로 평가됩니다.

분석에는 HuggingFace에 공개된 한국어 혐오 표현 분석 모델 smilegate-ai/kor_unsmile 이 사용되었습니다.

2. 사용 모델

  • 모델명: kor_unsmile
  • 제공: Smilegate AI
  • 분석 방식: 문장 단위 다중 혐오 카테고리 확률 추정
  • 출력 값: 각 카테고리별 0~1 범위의 확률 값

3. 결과 파일(CSV) 구조

혐오 카테고리별 확률 값

결과 CSV 파일에는 각 문장에 대해 다음과 같은 혐오 관련 카테고리 열이 포함됩니다. 각 값은 해당 문장이 특정 혐오 유형에 해당할 확률을 의미합니다.

  • 여성/가족
  • 남성
  • 성소수자
  • 인종/국적
  • 연령
  • 지역
  • 종교
  • 기타 혐오
  • 악플/욕설
  • clean

예를 들어 악플/욕설 = 0.46 인 경우, 해당 문장이 악플 또는 욕설에 해당할 가능성이 46%임을 의미합니다.

모든 카테고리 값은 하나의 문장에 대해 동시에 계산되며, 각 카테고리는 서로 독립적인 확률로 해석됩니다.
