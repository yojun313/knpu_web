음성 인식: Speech-to-Text (Whisper)

1. 분석 개요

음성 인식(STT: Speech-to-Text) 기능은 업로드된 오디오 파일(WAV/MP3/M4A/FLAC 등)을 입력으로 받아, 음성 내용을 문장 형태의 텍스트로 변환합니다. 변환 결과는 전체 발화 내용을 하나의 텍스트로 합친 형태로 제공되며, 분석 과정에서 언어를 지정하거나 자동 감지를 사용할 수 있습니다.

분석에는 OpenAI Whisper 계열 모델을 CTranslate2 기반으로 최적화한 Systran/faster-whisper-large-v3 모델이 사용되었습니다.

2. 사용 모델

  • 모델명: faster-whisper-large-v3
  • 기반 모델: OpenAI Whisper Large V3 계열
  • 제공: Systran (Hugging Face 공개)
  • 추론 엔진: CTranslate2 기반 최적화(고속 추론)
  • 분석 방식: 음성 → 텍스트(문장 단위 출력 후 병합)
  • 언어 지원: 다국어 지원 (한국어 포함)

3. 입력 파일(오디오) 형식

음성 인식은 다음과 같은 오디오 파일 포맷을 입력으로 받습니다. 파일 포맷이 지원 범위를 벗어나거나 손상된 경우 분석이 실패할 수 있습니다.

  • .wav
  • .mp3
  • .m4a
  • .flac
  • .ogg

파일 업로드 전, 오디오가 정상 재생되는지 확인해 주세요. 특히 매우 긴 음성 파일(예: 수 시간 분량)은 처리 시간이 길어질 수 있으며, 서버 환경에 따라 처리 중 제한이 발생할 수 있습니다.

4. 언어 선택 옵션

음성 인식은 다음 중 하나로 언어 처리를 수행합니다.

  • 한국어(기본값): ko
  • 자동 감지: 입력 음성으로부터 언어를 자동으로 추정
  • 기타 언어: 영어(en), 일본어(ja), 중국어(zh) 등

특정 언어로 고정하면(예: ko) 인식 안정성이 높아질 수 있으며, 음성에 여러 언어가 섞인 경우에는 자동 감지를 권장합니다.
